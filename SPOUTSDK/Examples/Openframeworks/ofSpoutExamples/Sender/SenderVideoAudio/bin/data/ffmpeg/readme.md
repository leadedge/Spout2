
FFmpeg.exe is required for the SenderAudioVideo example.

* Go to https://github.com/GyanD/codexffmpeg/releases
* Choose the "Essentials" build. e.g. "ffmpeg-8.1.2-essentials_build.zip" 
* Download the archive to a convenient folder.
* Unzip the archive and copy "bin\ffmpeg.exe" and "bin\ffprobe.exe"
  to the application "bin\data\ffmpeg" folder
  
The data folder also contains the "videos" and "fonts" folders
